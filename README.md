# 五臓六腑クエスト 〜経絡の秘宝〜

東洋医学(五臓六腑・経絡)を学べるレトロ風2D学習RPGです。
ブラウザだけで動きます — `index.html` をダブルクリックで起動してください。

## あそびかた

| キー | 操作 |
|---|---|
| 矢印キー / WASD | 移動 |
| Z / Enter / スペース | 話す・調べる・決定 |
| X / Esc | メニューを開く・にげる |

## ゲームの目的

病魔王が放った六淫の邪気(風・寒・暑・湿・燥・火)が五行の里を蝕んでいます。

1. **賢者に学ぶ** — 各地方にいる五臓六腑の賢者(肝・心・脾・肺・腎/胆・小腸・胃・大腸・膀胱・三焦+心包)から知識を教わる
2. **邪気とバトル** — 草むらで出会う邪気とはクイズで戦う。正解で攻撃、不正解でダメージ+解説表示
3. **祠の試練** — 東西南北+中央の祠で守護神の試練に合格し「五行の玉」を5つ集める
4. **病魔王を倒す** — 玉が揃うと北の城門が開く

## マップ(五行の配置)

- **東の森 = 木** … 肝・胆
- **南の赤土 = 火** … 心・小腸
- **中央の里 = 土** … 脾・胃(村に心包・三焦の賢者も)
- **西の岩場 = 金** … 肺・大腸
- **北の湖 = 水** … 腎・膀胱

## 学習資料(Xメニュー)

- 五行色体表(五臓・五腑・五官・五主・五志・五味・五色・五液・五華)
- 十二経脈の流注順序と奇経八脈
- 五臓・六腑のまとめ

## ヒント

- 村長に話しかけるとHPが全回復します
- 間違えても解説が出るので、負けを恐れず挑戦を(敗北ペナルティはHP半分で村に戻るだけ)
- 進行状況は自動セーブされます(ブラウザのlocalStorage)
