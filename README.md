# 五臓六腑クエスト

**～からだの国と失われた調和～**

東洋医学の五臓六腑を、会話・探索・問診バトルで学ぶスマートフォン縦画面向けWebゲームです。プロトタイプでは「心の城」と「肝の森」を最後まで遊べます。

## 公開版

https://lionemikosan-sketch.github.io/gozoroppu-quest/

## 起動方法

ES Modulesを使っているため、ローカルサーバーから起動します。このフォルダで次を実行し、表示されたURLをブラウザで開きます。

```bash
python3 -m http.server 8080
```

アクセス先：

```text
http://localhost:8080
```

## 実装済み

- 9:16のスマートフォン専用レイアウト（PCでは中央表示）
- タイトル、名前入力、つづきから、音ON/OFF
- 11地域を配置したワールドマップ
- 心の城、肝の森
- 各地域4つの探索ポイントと短い会話イベント
- 心「神明かくし」、肝「気滞のツタ」との問診クイズバトル
- 正解・不正解の解説、画面フラッシュ・揺れ・星・霧晴れ演出
- 28問のクイズデータとランダムな選択肢
- 五臓六腑図鑑、未解放シルエット、クイズ正解率
- 初期アイテム、主人公プロフィール、獲得した五行スキル
- `localStorage` による自動セーブ
- Web Audio APIによる短い効果音（音なしでも動作）
- タッチ操作、キーボードのEnter・Esc操作

## 遊び方

1. 「はじめから」を押し、主人公の名前を入力します。
2. マップの「心の城」を選びます。
3. 4つの探索ポイントを調べ、役割の記憶を集めます。
4. 問診バトルで3問正解すると地域をクリアできます。
5. 解放された「肝の森」も同じように攻略します。

画面下のメニューから、どうぐ・スキル・仲間・図鑑を確認できます。進行は操作のたびに自動保存されます。

## ステージの追加方法

1. `js/gameData.js` の `REGIONS` で対象地域に `implemented: true` を追加します。
2. 同ファイルの `STAGES` に、次の情報を持つステージを追加します。
   - `intro`：導入会話
   - `points`：探索ポイント（会話と報酬）
   - `bossIntro`：ボス前会話
   - `clearDialogue`：クリア後会話
   - `quizTags`：出題するクイズのタグ
   - `unlocks`：次に解放する地域ID
3. `js/app.js` の `finishStage` に、その章で解放する図鑑と五行スキルを追加します。

既存の `heart` または `liver` を複製すると、同じ流れの章を追加できます。

## クイズの追加方法

`js/quizData.js` の `QUIZZES` に次の形式で追加します。

```js
{
  id: "q29",
  text: "問題文",
  answer: "正解",
  choices: ["正解", "選択肢2", "選択肢3", "選択肢4"],
  category: "organ",
  tags: ["heart"],
  hint: "不正解時にも役立つ短い解説"
}
```

- `answer` は `choices` に必ず含めます。
- 選択肢の順番は出題時に自動で変わります。
- ステージの `quizTags` と同じタグを付けると、そのボス戦で出題されます。

## スマートフォンでの確認

PCとスマートフォンを同じWi-Fiに接続し、PCでローカルサーバーを起動します。PCのローカルIPが `192.168.1.20` の場合、スマートフォンで次を開きます。

```text
http://192.168.1.20:8080
```

SafariまたはChromeの縦向き表示で確認してください。ホーム画面へ追加すると、よりゲームらしく表示できます。
